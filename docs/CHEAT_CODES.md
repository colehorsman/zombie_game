# Zombie Game Cheat Codes

## Admin Testing Cheat Codes

These cheat codes are for testing and debugging purposes.

### Level Access
- **`UNLOCK_ALL`** - Unlock all levels (bypass progression)
  - Press keys in sequence: `U` → `N` → `L` → `O` → `C` → `K`
  - All account room doors will become accessible

### Power-Ups (Future)
- **`GOD_MODE`** - Invincibility

- **`INFINITE_AMMO`** - Unlimited projectiles
  - Press keys: `A` → `M` → `M` → `O`

### Game State
- **`SKIP_LEVEL`** - Mark current level as complete and return to lobby
  - Press keys: `S` → `K` → `I` → `P`

## How to Use Cheat Codes

1. During gameplay, simply press the keys in sequence
2. You'll see a message confirming the cheat code was activated
3. Cheat codes work in both lobby and level modes

1. **Sandbox** (577945324761) - Always unlocked
2. **Stage** (514455208804) - Unlocks after Sandbox complete
3. **Automation** (393582650665) - Unlocks after Stage complete
4. **WebApp** (240768036625) - Unlocks after Automation complete
5. **Production Data** (160224865296) - Unlocks after WebApp complete
6. **Production** (613056517323) - Unlocks after Production Data complete
7. **Org** (437154727976) - Final level

Complete a level by quarantining all zombies and returning to the lobby.
