## Mission Alignment with Sonrai Security

**Sonrai's Core Message:** "Secure your cloud with intelligent identity and data protection"

**Game's Educational Mission:**
- **Visibility** - Visualize unused identities as zombies, making the invisible visible
- **Control** - Demonstrate least privilege through purple shields and JIT access
- **Automation** - Show real-time API integration with actual quarantine actions
- **Intelligence** - Teach WHY security matters through contextual education

## Core Concept

**The Problem We're Solving:**
Cloud security is abstract and intimidating. Terms like "unused IAM identity" and "third-party access" don't resonate with most people. This game transforms complex security concepts into tangible, understandable gameplay.

**How It Works:**
- **Zombies** = Unused AWS identities (contractor accounts, old service roles, forgotten users)
- **Eliminating zombies** = Real Sonrai API calls that quarantine identities
- **Third-party entities** = External services with access to your cloud
- **Purple shields** = Protected/exempted identities (Sonrai, approved accounts, JIT-protected roles)
- **Quests** = Real security scenarios (service protection, JIT access, compliance)

**Educational Value:**
Every game mechanic teaches a security concept. Players learn by doing, not by reading documentation.

## Key Features

### Gameplay
- **Dual-mode gameplay** - Top-down lobby exploration + side-scrolling platformer levels
- **Real-time Sonrai API integration** - Every action triggers actual cloud security operations
- **Progressive difficulty** - Scales from beginner-friendly to expert-level challenges
- **Side quests** - Service Protection (hacker race) + JIT Access (auditor challenge)
- **Controller support** - Bluetooth/wired controllers for authentic retro feel

### Educational
- **Contextual learning** - Pop-up explanations teach concepts during gameplay
- **Interactive glossary** - In-game reference for all security terms
- **Real-world examples** - Breach stories between levels show why security matters
- **Visual metaphors** - Color-coded risk levels, purple shields for protection
- **Achievement system** - Rewards learning milestones, not just gameplay

### Technical
- **Performance optimized** - 60 FPS with 500+ entities via spatial grid
- **Save/load system** - Persistent progress with autosave
- **Difficulty modes** - Adjustable for different age groups and skill levels
- **Stats dashboard** - Shows real-world security impact of player actions
